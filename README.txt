The assignment consists of 3 simulations:

1. Image Generalization using a neighbourhood of 5
	Files: 	imageGen_5.ma
		execute the simulation using EXEC_imageGen_5.BAT
		draw the simulation using DRAW_imageGen_5.BAT
		colour palette : imageGen_5.pal
	Files generated:	imageGen_5.log
			imageGen_5.drw

2. Image Generalization using a neighbourhood of 9
	Files: 	imageGen_9.ma
		execute the simulation using EXEC_imageGen_9.BAT
		draw the simulation using DRAW_imageGen_9.BAT
		colour palette : imageGen_9.pal
	Files generated:	imageGen_9.log
			imageGen_9.drw


3. Image Generalization using a neighbourhood of 25
	Files: 	imageGen_25.ma
		execute the simulation using EXEC_imageGen_25.BAT
		draw the simulation using DRAW_imageGen_25.BAT
		colour palette : imageGen_25.pal
	Files generated:	imageGen_25.log
			imageGen_25.drw


The directory named 'Java' contains the java files to execute the java procedure that converts a .raw file to .MAP file format.
Also contains the original picture.bmp.
